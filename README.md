## 🚀 快速开始

如果您想快速了解如何使用此项目，请参考以下步骤：

1. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

2. **配置环境变量**：
   - 复制 `.env.example` 并重命名为 `.env`
   - 编辑 `.env` 文件，填写您的配置信息

3. **运行程序**：
   ```bash
   python CVE_PushService.py
   ```

详细的使用说明请查看 [docs/README.md](docs/README.md) 和 [docs/LOCAL_TEST_GUIDE.md](docs/LOCAL_TEST_GUIDE.md)。
